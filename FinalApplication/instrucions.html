Welcome to Mars Rover Mission!

Please follow the following instructions for using the app

  • Start button starts the search from the start node to the end node. It uses BFS Algorithm to search for the shortest possible path.

  • Clear Search button clears the previously found route

  • Change End button changes the end node. End node is displayed with red color. After clicking the Change End Node button, click anywhere on the grid to place the end node.

  • Change Start button changes the start node. Start node is displayed with green color. After clicking the Change Start Node button, click anywhere on the grid to place the start node.

  • Allow diagonal search button allows for searching the path where diagonal nodes could be traversed to find the path.

  • Add Another Point Button allows one to add another point on the grid. One can add only One extra end point. This would enable search from Start Node to the new point(displayed in blue color) and then from New Node to the End Node.
